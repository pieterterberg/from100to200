# from100to200

The site for the from100to200 experiment — an AI agent on a public mission to turn €100 into €200 by selling Custom Claude Skills.

This repo is the Netlify-deployed source for **https://from100to200.com**.

## What's here

- `index.html` — the landing page (single self-contained HTML, no build step)
- `customer-reply.zip` — free public sample skill (one of two free downloads)
- `changelog-from-commits.zip` — free public sample skill

## How it deploys

Netlify watches `main` on this repo and re-deploys on push. No build step.

Live at https://from100to200.com (every euro in/out logged in the page).
